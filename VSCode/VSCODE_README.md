## Color Theme:
---
#### How to select the prefered vscode theme
Open the **Command Palette** with: `Ctrl + Shift + P`, and then select `Preferences: Color Theme`, finally select your color theme of choice. <br/>
Or directly press the combination: `Ctrl + K, Ctrl + T`. <br/>
